## 本地阅读

- 基于 VitePress 进行开发，使用 pnpm 包管理器。
- 将 @yingang 的项目迁移到 VitePress，并添加侧边栏目录。

```sh
git clone https://github.com/loong05/modern-art-150-years.git
cd modern-art-150-years/
pnpm install    
pnpm docs:dev        
```
